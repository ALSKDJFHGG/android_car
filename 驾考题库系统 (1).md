# 移动端/考试

## GET 获取试题列表

GET /mobile/exam/list

> 返回示例

> 200 Response

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "list": {
      "paperId": 0,
      "paperName": "string",
      "subject": "string",
      "totalQuestions": 100,
      "examTime": 0
    }
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|Inline|

### 返回数据结构

状态码 **200**

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|» code|integer|true|none||响应码|
|» message|string|true|none||响应消息|
|» data|object|true|none||none|
|»» list|object|true|none||none|
|»»» paperId|integer|true|none||试题ID|
|»»» paperName|string|true|none||试卷名称|
|»»» subject|string|true|none||科目一/科目四|
|»»» totalQuestions|integer|true|none||题目数量|
|»»» examTime|integer|true|none||分钟|
